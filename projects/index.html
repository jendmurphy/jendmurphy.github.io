GEOG5995 Agent Based Model Course Practical

Click on the title to go to the repository for my first ever forage into the world of Python. Code here for an agent based model where agents move around an environment, nibbling away at data and building up stores. Some cooperative behaviour is included whereby nearby agents share their stores, contingent on a defined neighbourhood (a measure of proximity). The code is animated to allow the user to see the gradual errosion of data through the random movement of agents. This can be seen by a change in the colour of the underlying environment in the wake of each agent. The repository also includes a pdf UML diagram for the model.

The course was taught by Andy Evans of Leeds University.
